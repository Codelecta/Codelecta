# Codelecta

Codelecta is a static educational website built with HTML, CSS, and JavaScript. It provides easy navigation between pages for courses, services, contact, feedback, and user account management.

## Project Structure

- `index.html` - Main landing page
- `menu.js` - Navigation menu behavior
- `navBar.css`, `footer.css` - Shared styling for navigation and footer
- `aboutUs/` - About Us page and styles
- `contactPage/` - Contact page and styles
- `coursePage/` - Course listing and individual course pages
- `feedbackPage/` - Feedback page and styles
- `homePage/` - Home page and styles
- `loginPage/` - Login, signup, and password recovery pages and styles
- `paymentPage/` - Payment page and styles
- `policyPage/` - Policy page and styles
- `servicePage/` - Services page and styles

## Features

- Responsive course navigation
- Dedicated pages for services, contact, feedback, and policies
- Account entry flows including login, signup, and forgot password
- Clean, modular styling for page sections

## Running Locally

1. Open the `Codelecta` folder in your code editor.
2. Open `index.html` in a browser.
3. Navigate between pages using the site menu.
